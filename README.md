💻 Setup Instructions

```
# 1. Clone the repository
```

git clone https://github.com/Khalid-Saifullaha/Library-Management-API-with-Express-TypeScript-MongoDB.git

```
# 2. Install dependencies
```

npm install

```
# 3. Create .env file
```

MONGODB_URI=your_mongodb_connection_string
PORT=5000

```
 # 4. Run the project
```

npm run dev
